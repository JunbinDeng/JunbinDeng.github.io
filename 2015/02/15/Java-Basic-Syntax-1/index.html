Java 基础 -- 语法（一）

  • Java

Java基础语法将分成八个部分介绍：关键字，标识符，注释，常量与变量，运算符，语句，函数，数组。

1. 关键字

定义：被Java语言赋予了特殊含义的单词。

特点：关键字中所有字母都为小写。

作用于：

  • 定义数据类型：class, interface, byte, short, int, long, float, double, char, boolean, void；
  • 定义数据类型值：true, false, null；
  • 定义流程控制：if, else, switch, case, default, while, do, for, break, continue, return；
  • 定义访问权限修饰符：private, protected, public；
  • 定义类，函数，变量修饰符：abstract, final, static, synchronized；
  • 定义类与类之间关系：extends, implements；
  • 定义建立，引用及判断实例：new, this, super, instanceof；
  • 异常处理：try, catch, finally, throw, throws；
  • 包：package, import；
  • 其他修饰符：native, strictfp, transient, volatile, assert。

2. 标识符

定义：在程序中自定义的一些名称。

特点：由26个英文字母大小写，数字（0-9），符号（_$）组成。

合法标识符定义规则：

  • 数字不可以开头；
  • 不可以使用关键字。

提示：

  • Java严格区分大小写；
  • 为了提高阅读性，尽量起有意义的标识符名称。

多单词组合时的名称规范：

  • 包名：所有字母都小写。如：xxxyyyzzz；
  • 类名接口名：所有单词的首字母大写。如：XxxYyyZzz；
  • 变量名和函数名：第一个单词首字母小写，第二个单词开始每个单词首字母大写。如：xxxYyyZzz；
  • 常量名：多单词所有字母都大写，每个单词用下划线连接。如：XXX_YYY_ZZZ。

3. 注释

定义：用于注解说明程序的文字。

作用：提高代码的阅读性。

格式：

  • 单行注释
    • 格式：//注释文字
  • 多行注释
    • 格式：/*注释文字*/
  • 文档注释
    • 格式：/**注释文字*/

举例：

对于单行和多行注释，被注释的文字，不会被JVM（Java虚拟机）解释执行。

对于文档注释，是java特有的注释，其中注释内容可以被JDK提供的工具javadoc所解析，生成一套以网页文件形式体现的该程序的说明文档。

小贴士：
注释是一个程序员必须要具有的良好编程习惯。初学者编写程序可以养成先写注释再写代码的习惯，将自己的思想通过注释先整理出来，再用代码去体现。因为代码仅仅是思想的一种体现形式而已。
注释宜少而精，不宜多而滥。不写过于详细的注释，对显而易见的代码添加的注释，以及罗嗦的注释。

4. 常量与变量

4.1 常量

常量的定义：不能改变的数值。

常量的分类：

  • 整数常量：所有整数；
  • 小数常量：所有小数；
  • 布尔型常量：只有两个数值，true和false；
  • 字符常量：将一个数字，字母或者符号用一对单引号（’ ‘）标识；
  • 字符串常量：将一个或者多个字符用一对双引号（” “）标识；
  • null常量：只有一个数值就是null（空）。

4.1.1 进制

数运算的进位方法：

  • 二进制：0-1，满二进一；
  • 十进制：0-9，满十进一；
  • 八进制：0-7，满八进一；
  • 十六进制：0-9和A-F，满十六进一，用0x开头表示；

原码：所对应的正数二进制。

反码：所对应的正数二进制取反。

补码：所对应的正数二进制取反加1。是负数的二进制表现形式。

4.2 变量

变量的定义：内存中的一个存储区域。该区域有自己的名称（变量名）和类型（数据类型），该区域的数据可以在同一类型范围内不断变化。

作用：用来不断的存放同一类型的常量，并可以重复使用。

使用变量时注意：

  • 其作用范围（在一对{}之间有效）；
  • 初始化值。

定义变量的格式：

数据类型 变量名 = 初始化值;

小贴士：格式是固定的，记住格式，以不变应万变。

4.2.1 数据类型分类

注：Java语言是强类型语言，对于每一种数据都定义了明确的具体数据类型，在内存中分配了不同大小的内存空间。

4.2.2 类型转换

  • 如果一个操作数是long型，计算结果就是long型；
  • 如果一个操作数是float型，计算结果就是float型；
  • 如果一个操作数是double型，计算结果就是double型。

5.4 逻辑运算符

注：逻辑运算符用于连接布尔型表达式，在Java中不可以写成a < x < b，应该写成 x > a & x < b。

  • “&”表示左边无论真假，右边都进行运算；

  • “|”和“||”的区别同理。

6. 程序流程控制

程序流程控制包含：判断结构，选择结构，循环结构。

6.1 判断结构

6.1.1 if语句

格式1：

if (条件表达式) {
    执行语句;
}

格式2：

if (条件表达式) {
    执行语句;
} else {
    执行语句;
}

格式3：

if (条件表达式) {
    执行语句;
} else if (条件表达式) {
    执行语句;
......
} else {
    执行语句;
}

if语句特点：

  • 每一种格式都是单条语句。

  • 第二种格式与三元运算符的区别：

    • 三元运算符运算完要有值出现；
    • 好处是：可以写在其他表达式中。

  • 条件表达式无论写成什么样子，只看最终的结构是否是true或者false。

6.2 选择结构

6.2.1 switch语句

格式：

switch (表达式) {
    case 取值1:
        执行语句;
        break;
    case 取值2:
        执行语句;
        break;
    ......
    default:
        执行语句;
        break;
}

switch语句特点：

  • switch语句选择的类型只有四种：byte，short，int，char。
  • case之间与default没有顺序。先执行第一个case，没有匹配的case执行default。
  • 结束switch语句的两种情况：遇到break，或是执行到switch语句结束。
  • 如果匹配的case或者default没有对应的break，那么程序会继续向下执行，运行可以执行的语句，直到遇到break或者switch语句结束。

6.3 循环结构

循环结构的3个代表语句： while, do while, for。

6.3.1 while语句

格式：

while (条件表达式) {
    执行语句;
}

6.3.2 do while语句

格式：

do {
    执行语句;
} while (条件表达式);

do while特点是条件无论是否满足，循环体至少被执行一次。

6.3.3 for语句

格式：

for (初始化表达式; 循环条件表达式; 循环后的操作表达式) {
    执行语句;
}

注：

  • for里面的表达式运行的顺序：

    1. 初始化表达式，只读一次；
    2. 判断循环条件，为真就执行循环体；
    3. 执行循环后的操作表达式；
    4. 重复过程2到3，直到循环条件不满足为止。

  • while与for可以互换，区别在于，for为了循环而定义的变量在for循环结束后在内存中释放，而while循环使用的变量在循环结束后还可以继续使用。

  • 最简单无限循环格式：while(true) , for(;;)。无限循环存在的原因是并不知道循环多少次，而是根据某些条件，来控制循环。

6.4 其他流程控制语句

6.4.1 break语句

break（跳出）：结束整个循环体。

应用范围：选择结构和循环结构。

6.4.2 continue语句

continue（继续）：结束本次循环，继续下个循环。

应用范围：循环结构。

注：

  • 这两个语句离开应用范围，存在是没有意义的。
  • 这两个语句单独存在，下面都不可以有语句，因为执行不到。
  • 标号的出现，可以让这两个语句作用于指定的范围。
