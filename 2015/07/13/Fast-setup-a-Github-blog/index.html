添加 SSH key 到 GitHub

  1. 用记事本打开 C:\Users\你的系统用户名\id_rsa.pub 文件。内容为刚才生成的密钥，复制下来。
  2. 登陆 GitHub，点击右上角的 Account Settings -> SSH Public keys -> add another public keys。
  3. 把刚才复制的密钥粘贴到 key 文本框中，点击 Add key 就完成了。
